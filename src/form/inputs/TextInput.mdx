# TextInput

Component that handles text input. It automatically registers itself to the nearest wrapping `Form` component. (See "Inputs" page for more information)
It supports the following labels out of the box:
- `placeholder`
- `required`
- `minLength`
- `maxLength`
- `pattern`

<StoryContainer>
  <Form
    name="form"
    onSubmit={() => null}
    config={{
      useForm: { mode: 'onChange' }
    }}
  >
    <TextInput name="nickname" />
  </Form>
</StoryContainer>
